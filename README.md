📌 Project Overview

This project was developed to address the challenge of managing dispersed information across different areas of a department. The Database Center consolidates data from Mining, Processes, Erosions, Ombudsman Reports, and Public Complaints into a single, organized system.

By providing tools to create, search, update, and manage records, the system improves efficiency and reduces fragmentation of information. It reflects a practical approach to building structured, real-world data solutions.

The system includes a simple password-based access control to ensure that only authorized users can interact with the data.

🚀 Features
- Centralized Database System  
  Unified platform to manage multiple operational areas: Mining, Processes, Erosions, Ombudsman, and Reports.

- Modular Data Management  
  Each module allows:
  - Adding new records  
  - Listing stored information  
  - Searching and filtering data  
  - Editing existing entries  
  - Deleting records  

- Secure Access Control  
  The system is protected by a password to restrict access to authorized department members.

- Search Functionality  
  Enables quick retrieval of specific records across different modules.

- Structured Data Organization  
  Information is categorized by department, improving clarity and maintainability.

- Scalable Architecture  
  Designed to support the addition of new modules or data categories in the future.
